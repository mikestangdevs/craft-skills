# setup

One-time setup for a repository using the craft skills.

| Skill | What it does |
|---|---|
| [`setup-craft-skills`](setup-craft-skills/SKILL.md) | Run once per repo (optional but recommended). Scaffolds a `CONTEXT.md` — domain glossary + conventions + untouchables — that the craft skills read *if present* to match your codebase. The skills work without it. |
